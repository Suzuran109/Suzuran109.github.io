安装配件要求

Node.js

这是一个好用的跨平台的JavaScript运行环境

下载地址：https://nodejs.org/zh-cn

git

这是GitHub、Gitee上传or下载文件用的分布式版本控制系统

下载地址1：https://gitforwindows.org/

下载地址2：https://git-scm.com/downloads/win

（git用 git bash）

安装Hexo

不出意外，安装好Node.js和git，用git bash打开后应该就可以使用npm指令了，使用npm安装Hexo
